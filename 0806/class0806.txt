Bootstrap : 프레임워크
CSS랑 javascript 내용에 대한 태그 head태그 안에 복붙

script나 style태그가 head에 있을 때랑 body에 있을 때 차이
head에 있으면 빠름
body에 있으면 그 script나 style태그를 로딩하는데 시간이 걸림
head에만 많이 써도 그걸 로딩하는데도 오래걸려서 body 출력이 안될수도있음

디자인 같은 눈에 시각적으로 보여야하는 경우는 눈에 보여야하므로 head에 쓰는게 좋음

반응형 웹페이지 : 페이지크기가 기기에 따라 바뀔 수도 있고, 화면크기를 조절할 수 있다


docs > layout > grid
bootstrap은 화면을 12개로 쪼개놨다
class row
3개를 쓰면 균등하게 4칸씩 배정해준다
비율 조정도가능 대신 12는 맞춰줘야 꽉 차고 작으면 작게
크면 넘쳐서 다음줄로 보여진다
col-2
col-3
2:3으로 배정

col-md-2 : 창 크기가 md크기 이상인 경우 2차지하게 하겠다
col-md-10 : 창 크기가 md크기 이상인 경우 10차지하게 하겠다

창 어느 크기 기준
sm
md
lg
xl
xxl
fluid

d-sm-none : 크기가 sm보다 작아질 경우 안보이게 할 수도 있음
